# FullStack Sites — Company Portfolio Website

This is the official portfolio website for **FullStack Sites**, a one-stop web development agency that handles both front-end design and back-end systems for small businesses, startups, and organizations.

## Tech Stack

- **HTML5** — Semantic structure
- **Tailwind CSS** (CDN) — Utility-first styling
- **Alpine.js** (CDN) — Interactive components (mobile nav toggle)
- **Custom CSS** — Gradients, animations, hover effects
- **AI-generated images** — Hero banner and portfolio placeholders

## Sections

1. **Hero** — Catchy headline with CTA buttons
2. **Tech Badges** — Technology stack indicators
3. **Services** — Front-End Dev, Back-End Systems, Hosting & Deployment
4. **Process** — 4-step workflow (Discovery → Design → Build → Launch)
5. **Portfolio** — 3 placeholder project cards with AI-generated images
6. **Pricing** — 3 tiers (Landing Page $1.5k+, Business Site $3.5k+, Full Application)
7. **Contact** — Inquiry form with company info
8. **Footer** — Navigation links and copyright

## How to Update

### Content Changes
Simply edit `index.html` — all content is in one file. Each section is clearly commented.

### Image Changes
Replace images in the `images/` directory. Maintain the same filenames or update `src` attributes in `index.html`.

### Deploying Changes
Push to the `main` branch. GitHub Actions (or GitHub Pages) will auto-deploy from `main` or `gh-pages` branch.
